# Coworking Spaces

A simple directory of laptop-friendly cafés and coworking spots in Leuven.

## Local Development

```bash
bun install
bun run dev
```

## Add a Space

**Easiest:** [Open an issue](https://github.com/Pezmc/coworking-spaces/issues/new?template=suggest-space.yml) with the space details.

**Or:** Submit a PR — see [CONTRIBUTING.md](./CONTRIBUTING.md) for category definitions.
